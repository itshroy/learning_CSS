#  AdoptMe - Pet Adoption Landing Page

A simple and responsive pet adoption landing page built using HTML and CSS. The project showcases adoptable pets through visually appealing cards containing images, names, and descriptions to encourage pet adoption.

##  Overview

AdoptMe is a frontend web project designed to practice modern CSS layouts, card components, image styling, and responsive design principles. The page features pet profiles displayed in clean and attractive cards.

##  Features

* Responsive card-based layout
* Pet profile cards with images and descriptions
* Clean and minimal user interface
* Rounded image and card styling
* Beginner-friendly HTML and CSS project
* Easy to customize and extend

##  Technologies Used

* HTML5
* CSS3
